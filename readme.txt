==============! Important !==============
프로그램 실행 이전에 yolov5 실행을 위한 패키지 설치가 필요함
                [설치 방법]
1. venv 생성 후, 활성화

2. "pip install -r ./requirements.txt" 실행
=========================================
               [프로그램 실행 방법]
1. 영양성분표가 찍힌 사진을 사전에 준비

2. 준비한 사진의 경로를 인자로 main.py를 실행
   ex) python main.py ./img.jpg

3. 터미널에서 텍스트 형태의 결과 확인
=========================================
